This folder implements a GeneSys example module that allows the user to manipulate
protein examples previously downloaded from official PATRIC database at
https://www.bv-brc.org/

It requires the following packages:
    -> BV-BRC CLI 1.040
    Go to https://github.com/BV-BRC/BV-BRC-CLI/releases and
    install the .deb version in Ubuntu
    -> A Debian execution environment as a consequence of the previous requirement
    -> byopython for BLAST
